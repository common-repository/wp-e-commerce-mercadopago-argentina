Plugin para agregar el Gateway de MercadoPago a WP e-Commerce

== Description ==

### Integración con WP e-Commerce ###

-UI Fácil de usar
-Modo "Debug" para encontrar errores

El plugin es útil para todos los que desarrollamos en WordPress y no tenemos otras opciones más que MercadoPago y DineroMail en Argentina!! 


== Installation ==

Se requiere tener instalado el plugin WP eCommerce
1. Subir por FTP el archivo `mercado-pago.php` a `/wp-content/plugins/wp-eCommerce/wpsc-merchants`
2. Activar el plugin en el menu de plugins 
3. Ir al menu merchants de WP eCommerce , elegir MercadoPago y completar Client_id y Client_secret.

== Frequently Asked Questions ==

- Estoy en Debug Mode y dice que hay un error de autenticación
Fijate que no haya espacios al final del acc id, secret y token en la pantalla de configuracion del gateway de MercadoPago en las opciones de wp e-commerce
